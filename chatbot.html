  • What is PotatoBot?

    PotatoBot is an AI chatbot built to help you on your fitness journey! It is trained to answer anything, from "build me a 30 minute cardio workout", to "how are you doing today?" to "give me a list of core exercises". Ask away!


    How was this built?

    PotatoBot was built using OpenAI API. Currently, the bot is based on DaVinci-003 text model. Since requests to OpenAI's API cannot be made directly, we route our requests through Axios, an HTTP client.
